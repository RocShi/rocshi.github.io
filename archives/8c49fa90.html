使用 GitBook + 阿里云搭建个人书籍共享站点

发表于 2021-05-10 更新于 2022-05-12
分类于 建站 Valine：
使用 GitBook 将本地 markdown 文件渲染为静态 HTML，部署到阿里云服务器后通过 Nginx 进行网络代理供外部访问，适宜作为笔记、博客、书籍等内容的发布平台。

0 前言

在此前的几篇文章《使用 Hexo + GitHub Pages 搭建个人博客站点》、《将 Hexo 搭建的博客站点从 GitHub Pages 迁移到腾讯云 VPS》、《基于 Hexo 的个人博客站点高级自定义设置——使用 NexT 主题》中，我们使用 Hexo 静态博客系统打造了高度定制化的个人博客站点，并托管在远程云服务器，使用 Nginx 进行网络代理供外部访问，目前站点运行良好。

博客站点适宜发布各种类型的经验总结、个人感悟，但对于一些强内在关联属性的内容（例如一本书各个章节的读书笔记，LeetCode 题解等）则更适合通过其它的统一平台进行发布，例如 GitBook
