GitзүҲжң¬еә“з®ЎзҗҶжңҖдҪіе®һи·өвҖ”вҖ”д»Ҙи·ҹиёӘд»“еә“жҢҮе®ҡеҲҶж”ҜдёӢзҡ„д»Јз ҒдёәдҫӢ

еҸ‘иЎЁдәҺ 2019-05-18 жӣҙж–°дәҺ 2020-03-01
еҲҶзұ»дәҺ жҠҖжңҜ пјҢ Git Valineпјҡ
еңЁжң¬ең°еҗҢжӯҘиҝҪиёӘжҹҗдёӘиҝңзЁӢд»“еә“зҡ„жҹҗдёҖеҲҶж”ҜпјҲд»ҘdevelopеҲҶж”ҜдёәдҫӢпјүгҖӮ

1 иғҢжҷҜ

еңЁжң¬ең°еҗҢжӯҘиҝҪиёӘжҹҗдёӘиҝңзЁӢд»“еә“зҡ„жҹҗдёҖеҲҶж”ҜпјҲд»ҘdevelopеҲҶж”ҜдёәдҫӢпјүгҖӮ

2 й—®йўҳ

иө·еҲқе…ҲиҝӣиЎҢдәҶеҰӮдёӢж“ҚдҪңпјҡ

  • еҲӣе»әдёҖдёӘж–Үд»¶еӨ№пјҢеҲқе§ӢеҢ–дёәgitд»“еә“пјҡgit init
  • иҝӣиЎҢз”ЁжҲ·еҹәжң¬дҝЎжҒҜй…ҚзҪ®пјҡgit config
  • еҲӣе»әжң¬ең°д»“еә“зҡ„developеҲҶж”Ҝпјҡgit checkout -b
  • е…іиҒ”иҝңзЁӢд»“еә“gitең°еқҖпјҡgit remote add

жҺҘдёӢжқҘпјҢе°қиҜ•йҖҡиҝҮgit branch --set-upstream-toе°Ҷжң¬ең°д»“еә“developеҲҶж”ҜдёҺиҝңзЁӢд»“еә“developеҲҶж”ҜиҝӣиЎҢе…іиҒ”ж—¶жҠҘй”ҷпјҡdevelopеҲҶж”ҜдёҚеӯҳеңЁгҖӮдҪҝз”Ёgit branch -rеҸ‘зҺ°иҝңзЁӢд»“еә“еҲҶж”Ҝдёәз©әпјҢе…¶е®һдҪҝз”Ёgit branch -aеҸҜд»ҘеҸ‘зҺ°ж №жң¬зңӢдёҚеҲ°д»»дҪ•еҲҶж”ҜгҖӮ

3 и§ЈеҶі

еӣ дёәд»…д»…жҳҜе…іиҒ”дәҶиҝңзЁӢд»“еә“пјҢжң¬ең°д»“еә“е№¶жңӘиҺ·еҸ–иҝңзЁӢд»“еә“зҡ„дҝ®ж”№пјҲеҢ…жӢ¬еҲҶж”ҜдҝЎжҒҜпјүпјҢжүҖд»Ҙжң¬ең°зңӢдёҚеҲ°иҝңзЁӢд»“еә“зҡ„еҲҶж”ҜпјҢд№ҹе°ұж— жі•иҝӣиЎҢеҲҶж”Ҝе…іиҒ”ж“ҚдҪңгҖӮд»ҘеҸӮиҖғ1дёӯзҡ„е»әи®®пјҢеңЁе°Ҷжң¬ең°д»“еә“еҲҶж”ҜдёҺиҝңзЁӢд»“еә“еҲҶж”ҜиҝӣиЎҢе…іиҒ”еүҚпјҢеҸҜе…ҲйҖҡиҝҮgit fetch originе‘Ҫд»ӨжӢүеҸ–иҝңзЁӢд»“еә“зҡ„дҝ®ж”№пјҢиҝҷж ·дҫҝеҸҜиҺ·еҸ–иҝңзЁӢд»“еә“еҲҶж”ҜдҝЎжҒҜпјҢдҪҶиҝҳж— жі•зңӢеҲ°жң¬ең°еҲҶж”ҜгҖӮжңҖдҪіе®һи·өжҳҜдҪҝз”Ёgit pull origin developе‘Ҫд»ӨпјҲеүҚжҸҗжҳҜдҪ еҝ…йЎ»зҹҘйҒ“жғіиҰҒе°ҶиҝңзЁӢд»“еә“зҡ„е“ӘдёӘеҲҶж”Ҝе№¶е…Ҙжң¬ең°еҲҶж”ҜпјүпјҢдҫҝеҸҜзңӢеҲ°иҝңзЁӢд»“еә“зҡ„developеҲҶж”Ҝе’Ңжң¬ең°д»“еә“зҡ„developеҲҶж”ҜпјҲеҪ“еүҚеҲҶж”ҜпјүпјҢиҝӣиҖҢеҸҜиҝӣиЎҢеҲҶж”Ҝе…іиҒ”ж“ҚдҪңгҖӮ

4 з»“и®ә

жҖ»з»“дёҖдёӢжңҖдҪіе®һи·өпјҡ

еҪ“з„¶пјҢзӣҙжҺҘдҪҝз”Ёgit cloneд№ҹжҳҜеҸҜд»Ҙзҡ„пјҲиҝҷжүҚеә”иҜҘз®—жҳҜжңҖдҪіе®һи·өпјҒпјү

git checkout target_repo_branchе°ҶиҮӘеҠЁеңЁжң¬ең°еҲӣе»әдёҖдёӘеҗҚдёәtarget_repo_branchзҡ„еҲҶж”ҜпјҢе№¶е°Ҷе…¶дёҺиҝңзЁӢд»“еә“дёӯзҡ„target_repo_branchеҲҶж”ҜиҝӣиЎҢе…іиҒ”гҖӮжү§иЎҢе®Ңgit cloneеҗҺпјҢй»ҳи®ӨеңЁжң¬ең°д»“еә“еҲӣе»әзҡ„жҳҜmasterеҲҶж”ҜпјҢе№¶е°Ҷе…¶дёҺиҝңзЁӢд»“еә“зҡ„masterеҲҶж”ҜиҝӣиЎҢе…іиҒ”пјҢдҪҶйҖҡеёёжғ…еҶөдёӢпјҢmaserеҲҶж”Ҝд»…з”ЁдәҺзүҲжң¬йҮҠж”ҫпјҢжҲ‘д»¬е№¶дёҚдјҡзӣҙжҺҘеҗ‘materеҲҶж”ҜжҺЁйҖҒдҝ®ж”№пјҢиҖҢжҳҜеңЁе…¶е®ғеҲҶж”ҜпјҲеҰӮdevelopеҲҶж”Ҝпјүе®ҢжҲҗејҖеҸ‘дёҺи°ғиҜ•еҗҺпјҢеҶҚйҖҡиҝҮgit mergeе°ҶеҲҶж”ҜдёҠзҡ„е·ҘдҪңе№¶е…ҘmasterеҲҶж”ҜгҖӮ
